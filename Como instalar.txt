# 📖 Tutorial de Instalação do DarknessBot  

Este guia ensina como instalar e configurar o DarknessBot no Windows, Linux e Termux (Android).  

---

## 🖥️ Instalação no Windows  

### 1️⃣ Instalar o Node.js  
- Baixe e instale o [Node.js](https://nodejs.org/) (versão LTS recomendada).  
- Durante a instalação, marque a opção para adicionar o Node.js ao PATH.  

### 2️⃣ Instalar o Git  
- Baixe e instale o [Git](https://git-scm.com/downloads).  

### 3️⃣ Baixar o DarknessBot  
Abra o Prompt de Comando (CMD) e execute:  
git clone https://github.com/SeuRepositorio/DarknessBot.git
cd DarknessBot
### 4️⃣ Instalar as Dependências  
npm install
### 5️⃣ Configurar o Bot  
- Abra o arquivo config.js e edite as opções conforme necessário.  

### 6️⃣ Iniciar o Bot  
node bot.js
---

## 🐧 Instalação no Linux  

### 1️⃣ Instalar Dependências  
Abra o terminal e execute:  
sudo apt update && sudo apt upgrade -y
sudo apt install nodejs npm git -y
### 2️⃣ Baixar o DarknessBot  
git clone https://github.com/SeuRepositorio/DarknessBot.git
cd DarknessBot
### 3️⃣ Instalar as Dependências  
npm install
### 4️⃣ Configurar o Bot  
- Edite o arquivo config.js para ativar ou desativar funções.  

### 5️⃣ Iniciar o Bot  
node bot.js
> Dica: Para rodar o bot em segundo plano, use:  
nohup node bot.js &
---

## 📱 Instalação no Termux (Android)  

### 1️⃣ Instalar o Termux  
- Baixe e instale o [Termux](https://f-droid.org/packages/com.termux/).  

### 2️⃣ Atualizar e Instalar Dependências  
Abra o Termux e execute:  
pkg update && pkg upgrade -y
pkg install nodejs git -y
### 3️⃣ Baixar o DarknessBot  
git clone https://github.com/SeuRepositorio/DarknessBot.git
cd DarknessBot
### 4️⃣ Instalar as Dependências  
npm install
### 5️⃣ Configurar o Bot  
- Abra e edite o arquivo config.js para personalizar as funções.  

### 6️⃣ Iniciar o Bot  
node bot.js
> Dica: Para manter o bot rodando mesmo ao fechar o Termux, use:  
nohup node bot.js &
---

Agora o DarknessBot está instalado e funcionando! 🎮🔥
